# Simple Location Tracker

A **web-based location tracker** that logs your current location in real-time and displays the **address** along with a timestamp.  

This project uses **HTML, CSS, and JavaScript** and relies on the **OpenStreetMap Nominatim API** for reverse geocoding. No backend is required.  

---

## Features

- Track your **live location** with browser geolocation API  
- Convert **latitude and longitude** into **human-readable addresses**  
- Display **timestamped logs** of all location updates  
- Fully **responsive design**, works on mobile and desktop  
- No external map library needed  

---

## How to Use

1. **Clone or download** this repository.  
2. Open the `index.html` file in your browser.  
3. Click the **"Start Tracking"** button.  
4. Allow your browser to access your location.  
5. Your **latitude, longitude, and address logs** will appear below the button.  

---

## Technologies Used

- **HTML5** – page structure  
- **CSS3** – styling and layout  
- **JavaScript** – geolocation, reverse geocoding, and logging  
- **OpenStreetMap Nominatim API** – convert coordinates to addresses  

---

## Notes

- Location tracking requires **user permission**.  
- Works best on browsers that support the **Geolocation API**.  
- Logs are not stored permanently; they reset when you **refresh the page**.
